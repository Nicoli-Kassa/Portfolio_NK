## Instalação

Para executar o projeto localmente, siga os passos abaixo:

```bash
# Clone o repositório
git clone https://github.com/Nicoli-Kassa/Portfolio_NK.git

# Entre na pasta do projeto
cd Portfolio_NK

# Instale as dependências
npm install

# Execute o projeto
npm run dev
```
